Python Library

docluster library will provide all essential up-to-date NLP/ML models, as well as pipelining tools and super easy visualization features. It alsoprovides access to dpm and download pre-trained models and training corpus.

Docluster Package Manager (dpm)

dpm is a package manager that enables community share their NLP models or training corpora with the rest of the world. There is no need to search for corpus nor need to train a base model for eight hours to start your own work.

Playgrounds

Playgrounds provide an easy test for any model inside dpm. Anyone can try pre-trained Word2Vec or sentiment analysis on your own data. If you are please with the results, you can incorprate the model in your build with the docluster Python library and dpm.
